📖 What Is This App?

Welcome to Shkolla Digjitale — the #1 programming school in Kosovo for ages 8-18! Ask questions about the school, but you must earn tokens by beating mini-games first! 🎮

🎯 How It Works

  • 1. Pick your character (each has a special power!)
  • 2. Ask questions about Digital School
  • 3. Run out of tokens? Play a mini-game!
  • 4. Beat the game → earn tokens + points + coins
  • 5. Use tokens to ask more questions!
